Gate Operations

Gate type-system hierarchy

Abstraction of types for representing gates.

QuantExQASM.GateOps.AGate — Type

Abstract Gate type

source
QuantExQASM.GateOps.AGateLabel — Type

Abstract Gate Label, for unique gates

source
QuantExQASM.GateOps.AGateCall — Type

Abstract Gate Call, for tracking Gate labels applied to specific qubits

source

GateLabel and GateCall

Specific gate instance in a circuit, applied to a given (set of) qubit(s).

QuantExQASM.GateOps.GateLabelP — Type

Parametric Gate label. Tracks the gate symbol (:x,:y,:z, etc), and arbitrary parameters

source
QuantExQASM.GateOps.GateCall1P — Type

Parametric single qubit Gate call. Has a GateLabelP, target qubit index and register label

source
QuantExQASM.GateOps.GateCall2P — Type

Parametric two qubit Gate call. Has a GateLabelP, control and target qubit indices, register label, and base gate (assumes controlled U).

source
QuantExQASM.GateOps.GateCallNP — Type

Parametric n-qubit Gate call. Has a GateLabelP, control vector indices, target qubit index, register label, and base gate.

source
QuantExQASM.GateOps.Gate — Type

Wrap of Matrix to fit type system hierarchy
